# qwen3-30b-a3b real-provider evaluation — 2026-07-22

## Scope

- Provider: Runpod OpenAI-compatible endpoint
- Model: `qwen3-30b-a3b`
- Prompt version: `narrative-prompt.v3`
- Context version: `narrative-context.v2`
- Dialogue schema: `narrative-dialogue.v8`
- Evaluated cases: Player agency, clarification non-advancement, signal counter-example, signal positive example
- Secret handling: endpoint, API key, prompts, player text, raw narratives, raw provider responses, and request IDs were not recorded

## Results

The first cold-start-tolerant run used two repetitions per case (8 runs). It produced a 12.5% schema success rate, 0% quality pass rate, a 12.5% signal misfire rate, and 134,470 ms average latency among successful structured responses. The long first response is consistent with the expected cold-start behavior.

Qwen3 thinking was then disabled for the Runpod vLLM request so reasoning would not consume the structured-output completion budget. A follow-up run (4 runs) improved to:

- Schema success rate: 50%
- Quality pass rate: 25%
- Signal misfire rate: 25%
- Average latency among schema-successful responses: 6,396.5 ms
- Input tokens reported for successful responses: 10,579
- Output tokens reported for successful responses: 520

A targeted diagnostic run showed one valid clarification response and one Player-agency schema failure at `$.body`. This confirms the remaining failures are not only cold-start failures and remain nondeterministic.

## Disposition

**Not approved for production Narrative generation in the evaluated configuration.** Disabling Qwen3 thinking materially improved latency and structured-output reliability, but the observed 50% schema success and 25% quality pass rates are below the evaluator's default 80% schema-success gate. Signal false positives also remain unacceptable for progression commits.

Before approval, rerun multiple repetitions after changing the model, serving configuration, constrained-decoding support, or prompt/schema compatibility. Do not lower the production acceptance threshold merely to make this model pass.
